## Bugs Fixed

### Bug: Admin overview KPI "Cobrado" showing cents as euros
- **Where**: `src/pages/admin/AdminProjectDetail.tsx` line ~643
- **Problem**: `totalRevenue` is stored in cents (e.g. 495000 for 4950€). The overview KPI displayed `totalRevenue.toLocaleString()` directly, showing "495.000 €" instead of "4950 €".
- **Fix**: Changed to `(totalRevenue / 100).toLocaleString('es-ES')`.
- **Impact**: Admin was seeing wildly incorrect "Cobrado" figures in every project's overview tab.

### #1035 — Admin payment request creation in project payments tab
- **File**: `src/pages/admin/AdminProjectDetail.tsx`
- **What was added**:
  - "Crear solicitud de pago" button (emerald, with Plus icon) in the payments tab header
  - Toggle form with animated entry (motion.div): amount input (€), payment type dropdown (full/deposit/final/maintenance)
